# Ativações em Redes Neurais (From Scratch)

Este laboratório foca na implementação manual dos componentes fundamentais de uma rede neural profunda utilizando apenas **Python** e **NumPy**.

## 🚀 O que foi implementado:
- **Inicialização Dinâmica:** Criação de redes com N camadas e nós configuráveis.
- **Função Sigmóide:** Implementação matemática da ativação e sua derivada.
- **Forward Propagation:** Fluxo de dados através das somas ponderadas e ativações.
- **Visualização:** Gráficos de barras demonstrando a ativação de cada neurônio em diferentes camadas.

## 🛠️ Tecnologias
- Python 3.x
- NumPy
- Matplotlib
