# 📊 Dashboard de Vendas em Excel

Este projeto consiste na criação de um dashboard de vendas desenvolvido no Microsoft Excel, com foco na organização, visualização de dados e apoio à tomada de decisão baseada em dados.

## 🎯 Objetivo
Transformar dados brutos de vendas em informações visuais claras e intuitivas, permitindo analisar o desempenho comercial de forma rápida e eficiente.

## 🗂 Estrutura do Arquivo
O arquivo Excel está organizado da seguinte forma:

- **Assets**: Elementos visuais e auxiliares do dashboard
- **Bases**: Dados brutos utilizados na análise
- **Cálculos**: Métricas, agregações e tabelas auxiliares
- **Dashboard**: Visualização final com gráficos e indicadores

Essa separação segue boas práticas de Business Intelligence no Excel.

## 📈 Funcionalidades do Dashboard
- Visualização clara do desempenho de vendas
- Organização estruturada dos dados
- Apoio à análise e tomada de decisão
- Layout intuitivo e profissional

## 🛠️ Tecnologias Utilizadas
- Microsoft Excel
- Tabelas dinâmicas
- Gráficos
- Fórmulas e funções do Excel

## ▶️ Como Reproduzir o Projeto
1. Faça o download do arquivo `.xlsx`
2. Abra no Microsoft Excel (versão 2016 ou superior recomendada)
3. Navegue até a aba **Dashboard**
4. Utilize os filtros e visualizações disponíveis para análise

## 📁 Arquivo
- `Dashboard de Vendas.xlsx`
