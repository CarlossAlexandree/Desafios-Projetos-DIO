# Dashboard Gerencial para Tomada de Decisões Com Power BI

## 📋 Descrição do Projeto
Este repositório apresenta o Dashboard Gerencial para Tomada de Decisões, um projeto focado na convergência entre análise de dados robusta e User Experience (UX).

O desenvolvimento priorizou a clareza cognitiva e a eficiência analítica, resultando em um Relatório Financeiro a partir de dados modelado em 3 páginas dinâmicas e totalmente integradas

## 🛠️ Tecnologias e Conceitos Aplicados
* **Power BI**: Desenvolvimento de dashboards e tratamento de dados[cite: 62].
* **Design UX/UI**: Aplicação de princípios de posicionamento, contraste e proporção áurea[cite: 67, 68, 69].
* **Navegabilidade**: Criação de menus e botões para facilitar a exploração dos dados pelo usuário final[cite: 75, 78].

## 📈 Funcionalidades Implementadas
1.  **Análise de Vendas e Lucro**: Visualizações dinâmicas com foco em tendências[cite: 58].
2.  **Matriz de Descrição**: Detalhamento técnico para conferência de valores[cite: 59].
3.  **Segmentação de Dados**: Filtros otimizados para rápida filtragem por período e categoria[cite: 70].
4.  **Botões de Navegação**: Menus em cada página que destacam a visualização selecionada[cite: 77, 78].

## 🚀 Integração Cloud
Como este projeto visa gerar valor real e não apenas teórico:
* **Publicação**: O arquivo `.pbix` deve ser publicado no Power BI Service para acesso via web/mobile.
* **Integração Cloud**: Recomenda-se a conexão com fontes de dados em nuvem (como Azure SQL ou OneDrive) para permitir a atualização automática (Scheduled Refresh).
* **GitHub**: O versionamento é feito via arquivo físico, mas a documentação foca no processo de transformação de dados e experiência do usuário.
